# Обновления DeepAgents GUI

## ✅ Реализованные изменения

### 1. Расширенное окно настроек (SettingsDialog)

**Новые вкладки:**
- **LM Studio** - базовые настройки подключения
- **Инструменты** - управление инструментами агента
- **Безопасность** - настройки безопасности выполнения команд

### 2. Управление инструментами

**Возможности:**
- ✅ Включение/выключение файловой системы
- ✅ Включение/выключение веб-поиска (DuckDuckGo)
- ✅ Включение/выключение математики (SymPy)
- ✅ Включение/выключение обработки изображений (Pillow)
- ✅ Включение/выключение командной строки

**Автоматическое определение доступности:**
- Инструменты показываются как недоступные, если пакеты не установлены
- Отображается предупреждение о необходимом пакете

### 3. Настройки безопасности

**Режимы доступа к командной строке:**

#### Ограниченный режим (по умолчанию)
- Разрешены только команды из белого списка
- Белый список настраивается пользователем
- По умолчанию: `ls, dir, cat, head, tail, pwd, echo, find, grep`

#### Полный доступ (⚠ опасно)
- Разрешено выполнение ЛЮБЫХ команд
- Требуется явное подтверждение пользователя
- Рекомендуется только для доверенных моделей

### 4. Интеграция с основным приложением

**Метод `_get_configured_tools()`:**
- Динамически создаёт набор инструментов на основе настроек
- Передаёт параметры безопасности в инструмент `execute_command`
- Логирует все изменения для отладки

**Метод `_process_message()`:**
- Использует новый метод получения инструментов
- Учитывает настройки безопасности при выполнении

## 🔧 Как использовать

### 1. Открыть настройки
Нажмите кнопку **"⚙ Settings"** в главном окне

### 2. Управление инструментами
Перейдите на вкладку **"Инструменты"**:
- Включите/выключите нужные инструменты переключателями
- Недоступные инструменты показываются серым с предупреждением

### 3. Настройка безопасности
Перейдите на вкладку **"Безопасность"**:

**Для ограниченного режима:**
1. Убедитесь, что "Полный доступ" выключен
2. Отредактируйте список разрешённых команд
3. Введите команды через запятую (например: `ls, dir, pwd, cat`)

**Для полного доступа:**
1. ⚠ ВНИМАНИЕ: прочитайте предупреждение!
2. Включите переключатель "Полный доступ"
3. Список команд станет недоступен для редактирования
4. Все команды будут разрешены

### 4. Сохранить настройки
Нажмите **"Сохранить"** для применения изменений

## 📋 Примеры использования

### Безопасное использование (по умолчанию)
```
✅ Разрешено: ls, dir C:\, cat file.txt, pwd
❌ Запрещено: rm -rf, del /s, format C:
```

### Полный доступ (только для доверенных моделей!)
```
✅ Разрешено: любые команды включая rm, del, pip install
⚠ Риск: модель может выполнить опасные команды
```

## 🛡️ Рекомендации по безопасности

1. **Используйте ограниченный режим по умолчанию**
2. **Включайте полный доступ только если:**
   - Доверяете модели (проверенная локальная модель)
   - Понимаете риски
   - Работаете в изолированной среде (VM, контейнер)

3. **Никогда не включайте полный доступ для:**
   - Облачных моделей (OpenAI, Anthropic и др.)
   - Непроверенных открытых моделей
   - Продакшен-сред с важными данными
