```

```

## 🐛 ЧАСТЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ

### ❌ "Python не найден"

**Решение:**
1. Установите Python 3.10+ с https://www.python.org/downloads/
2. При установке отметьте **"Add Python to PATH"**
3. Перезапустите терминал

### ❌ "Папка 'deepagents' не найдена"

**Причина:** Запущено не из той папки

**Решение:**
```bash
cd D:\AIAgent\Deep_Agents_GUI
1_install_fixed.bat
```

### ❌ Тесты не проходят (ModuleNotFoundError)

**Решение:**
1. Убедитесь, что в папке проекта
2. Проверьте наличие `deepagents/__init__.py`
3. Перезапустите `1_install_fixed.bat`

### ❌ Окно закрывается сразу

**Решение:**
1. Запустите от имени администратора
2. Проверьте антивирус (может блокировать)
3. Смотрите логи в `install_log.txt`
