## Что такое Deep Agents GUI?

**Deep Agents GUI** — это графический интерфейс для управления AI-агентами на базе LangChain/LangGraph с возможностями:

### 🔧 Управление инструментами
- Динамическое включение/выключение инструментов
- Визуализация уровней риска (safe/review/dangerous)
- Редактирование описаний и параметров

### 🤖 Агент на базе LangGraph
- StateGraph с явными узлами и рёбрами
- Human-in-the-loop для опасных операций
- Сохранение состояния между сессиями

### 📊 Визуализация
- Интерактивная схема графа агента
- Пошаговая трассировка выполнения
- Интеграция с LangSmith для отладки

### 🔒 Безопасность
- Валидация всех входных данных
- Подтверждение опасных операций
- Защита от path traversal и injection

## Установка и настройка

### Шаг 1: Проверка системных требований

| Компонент | Требуется | Как проверить |
|-----------|-----------|---------------|
| ОС | Windows 10+ / macOS 10.15+ / Ubuntu 20.04+ | — |
| Python | 3.10 - 3.12 | `python --version` |
| RAM | 4 ГБ минимум | — |
| Интернет | Обязательно | — |

### Шаг 2: Установка зависимостей

**Windows:**
```bash
1_install.bat
```

**macOS/Linux:**
```bash
chmod +x *.sh
bash 1_install.sh
```

**Что происходит:**
- Создаётся виртуальное окружение `venv`
- Устанавливаются ~15 пакетов (LangChain, LangGraph, pytest и др.)
- Время: 3-5 минут

### Шаг 3: Настройка API ключей

**Вариант A: Через скрипт (рекомендуется)**
```bash
3_setup_env.bat
```

**Вариант B: Вручную**
1. Скопируйте `.env.example` в `.env`
2. Откройте `.env` в текстовом редакторе
3. Заполните ключи:

```ini
# OpenAI API Key (ОБЯЗАТЕЛЬНО)
OPENAI_API_KEY=sk-proj-xxxxxxxxxxxxxxxxxxxxxxxx

# LangSmith API Key (опционально)
LANGSMITH_API_KEY=lsv2_xxxxxxxxxxxxxxxxxxxxxxxxxxxx
LANGSMITH_TRACING=true
```

**Где получить ключи:**
- OpenAI: https://platform.openai.com/api-keys
- LangSmith: https://smith.langchain.com (опционально)

### Шаг 4: Проверка системы

```bash
2_run_tests.bat
```

**Ожидаемый результат:**
```
============================== 40 passed in 8.54s ==============================
[OK] ВСЕ ТЕСТЫ ПРОЙДЕНЫ УСПЕШНО!
```

## Основные возможности

### 1. Управление инструментами

**Панель Tool Manager:**
- ✅ Включать/выключать инструменты
- 📊 Видеть уровень риска каждого инструмента
- ✏️ Редактировать описания

**Уровни риска:**
| Уровень | Значок | Описание |
|---------|--------|----------|
| `safe` | 🟢 | Безопасные операции (чтение файлов, поиск) |
| `review` | 🟡 | Требуют подтверждения (запись файлов) |
| `dangerous` | 🔴 | Требуют особого подтверждения (удаление, команды) |

### 2. Human-in-the-Loop

При вызове инструмента с риском `review` или `dangerous`:

1. ⏸️ Выполнение приостанавливается
2. 💬 Появляется диалог подтверждения
3. ✅ Вы выбираете "Разрешить" или "Отклонить"
4. ▶️ Агент продолжает работу

### 3. Визуализация графа

**Graph Visualizer показывает:**
- 🔵 Узлы (nodes): agent, human_review, execute_tools, end
- ➡️ Рёбра (edges): условия переходов
- 🟢 Активный узел (подсвечивается при выполнении)

### 4. Трассировка выполнения

**Execution Trace Viewer:**
- 📝 История вызовов инструментов
- 📥 Входные аргументы
- 📤 Результаты выполнения
- ⏱️ Время выполнения каждого шага

### 5. Сохранение состояния

**Checkpointing:**
- 💾 Автоматическое сохранение после каждого шага
- 🔄 Возможность восстановления сессии
- 📁 Список предыдущих диалогов

## Решение проблем

### ❌ "Python не найден"

**Причина:** Python не установлен или не добавлен в PATH

**Решение:**
1. Скачайте Python 3.10+ с https://python.org
2. При установке отметьте **"Add Python to PATH"**
3. Перезапустите терминал

### ❌ "OPENAI_API_KEY не установлен"

**Причина:** Файл `.env` пустой или не создан

**Решение:**
```bash
3_setup_env.bat
# Затем отредактируйте .env и добавьте ключ
```

### ❌ "ModuleNotFoundError: No module named 'langchain'"

**Причина:** Зависимости не установлены

**Решение:**
```bash
# Активируйте venv вручную
venv\Scripts\activate  # Windows
source venv/bin/activate  # macOS/Linux

# Переустановите зависимости
pip install -r requirements.txt
```

### ❌ Тесты не проходят

**Возможные причины:**
1. Нет интернета
2. Неверный API ключ
3. Закончились кредиты на OpenAI

**Диагностика:**
```bash
5_quick_test.bat
# Смотрите детальные логи ошибки
```

### ❌ GUI не открывается / пустое окно

**Причина:** Проблемы с tkinter

**Решение для Windows:**
1. Переустановите Python
2. Выберите опцию "tcl/tk support"

**Решение для Linux:**
```bash
sudo apt-get install python3-tk
```

**Решение для macOS:**
```bash
brew install python-tk
```

## FAQ

### Q: Сколько стоит использование?

**A:** Вы платите только за API-вызовы к OpenAI по тарифам вашего аккаунта. Само приложение бесплатное.

### Q: Можно ли использовать без OpenAI?

**A:** currently поддерживается только OpenAI. Планируется добавление Anthropic и локальных моделей.

### Q: Где хранятся мои данные?

**A:** Все данные хранятся локально в папке проекта. Чекпоинты сохраняются в SQLite БД.

### Q: Как включить трассировку LangSmith?

**A:** Добавьте в `.env`:
```ini
LANGSMITH_API_KEY=ваш_ключ
LANGSMITH_TRACING=true
```

### Q: Можно ли добавить свои инструменты?

**A:** Да! Создайте класс наследник от `SafeBaseTool` в модуле `deepagents/tools.py`.

### Q: Как экспортировать конфигурацию агента?

**A:** Используйте кнопку "Экспорт" в GUI или файл `config_export.json` создастся автоматически.
