**Минимальные требования:**

- ✅ ~500 МБ свободного места

### Шаг 4: Настройка для LM Studio

Для работы с **локальными моделями** через LM Studio:

#### 1. Установите LM Studio
- Скачайте с https://lmstudio.ai/
- Установите модель (Llama 3, Mistral, Qwen и т.д.)

#### 2. Запустите локальный сервер
1. Откройте LM Studio
2. Перейдите в "Local Server"
3. Выберите модель
4. Нажмите "Start Server"
5. Порт: `http://localhost:1234`

#### 3. Настройте .env файл

Запустите:
```batch
3_setup_env.bat
```

Или создайте `.env` вручную:

```bash
# LM Studio - БЕЗ API КЛЮЧЕЙ!
OPENAI_API_KEY=lm-studio
OPENAI_API_BASE=http://localhost:1234/v1
OPENAI_MODEL_NAME=ваша-модель

# LangSmith (опционально, отключено)
LANGSMITH_TRACING=false
LANGSMITH_API_KEY=
LANGSMITH_PROJECT=deep-agents-gui-local
```

**Важно:** 
- `OPENAI_API_KEY` может быть любым (LM Studio не проверяет)
- `OPENAI_API_BASE` должен совпадать с портом LM Studio
- `OPENAI_MODEL_NAME` должно соответствовать названию модели

## 🐛 Решение распространённых проблем

### ❌ "Python не найден"

**Причина:** Python не установлен или не добавлен в PATH

**Решение:**
```bash
# Проверьте установку
python --version

# Если не найдено:
# 1. Переустановите Python с галочкой "Add to PATH"
# 2. Или добавьте вручную в системные переменные:
C:\Users\ВашПользователь\AppData\Local\Programs\Python\Python312
```

---

### ❌ "ModuleNotFoundError: No module named 'deepagents'"

**Причина:** Скрипт запущен не из корневой папки проекта

**Решение:**
```bash
# Перейдите в папку проекта
cd D:\AIAgent\Deep_Agents_GUI

# Проверьте наличие папки deepagents
dir deepagents

# Затем запустите установку
1_install_fixed.bat
```

---

### ❌ "Не удалось создать venv"

**Причина:** Проблемы с правами доступа или повреждённый Python

**Решение:**
```bash
# Удалите старый venv
rmdir /s venv

# Создайте новый
python -m venv venv

# Если ошибка сохраняется:
# 1. Запустите от имени администратора
# 2. Проверьте антивирус (может блокировать создание файлов)
```

---

### ❌ Тесты не проходят (0 passed, 18 failed)

**Причина:** Модули deepagents не найдены в PYTHONPATH

**Решение:**
```bash
# 1. Убедитесь, что находитесь в корневой папке
cd D:\AIAgent\Deep_Agents_GUI

# 2. Проверьте наличие папки deepagents
dir deepagents

# 3. Проверьте __init__.py
type deepagents\__init__.py

# 4. Переустановите зависимости
1_install_fixed.bat

# 5. Запустите тесты вручную
call venv\Scripts\activate
python -m pytest tests/ -v
```

---

### ❌ "pip install завершается без вывода"

**Причина:** Конфликт версий или проблемы с сетью

**Решение:**
```bash
# 1. Очистите кэш pip
pip cache purge

# 2. Обновите pip
python -m pip install --upgrade pip

# 3. Установите с флагом verbose
pip install -v -r requirements.txt

# 4. Если не помогает, установите вручную:
pip install langchain langchain-core langchain-openai langgraph langsmith
pip install pydantic python-dotenv pytest pytest-asyncio
pip install customtkinter networkx matplotlib httpx
pip install duckduckgo-search sympy Pillow
```

---

### ❌ GUI не запускается

**Проверьте логи:**
```bash
call venv\Scripts\activate
python deepagents_gui.py 2>&1 | tee gui_error.log
```

**Возможные причины:**
- Не установлен tkinter (обычно идёт с Python)
- Конфликт версий customtkinter
- Проблемы с графическим драйвером

**Решение:**
```bash
# Переустановите GUI библиотеки
pip uninstall customtkinter tkinter-tooltip Pillow
pip install customtkinter==5.2.0 tkinter-tooltip Pillow
```

## 💡 Советы для LM Studio

1. **Выбор модели**: Для начала используйте Llama-3-8B или Mistral-7B
2. **Контекст**: Установите context size >= 8192 для лучшей работы
3. **GPU Offload**: Включите максимальный GPU offload для скорости
4. **Температура**: В GUI установите temperature=0.7 для баланса креативности
