## 🔧 Функционал

### Главное окно
- **Чат** - общение с AI агентом
- **Инструменты** - управление доступными инструментами
- **Агенты** - создание и управление суб-агентами
- **Статистика** - метрики системы
- **Логи** - системные логи

### Менеджер инструментов (ToolManager)
- 9 встроенных инструментов в 6 категориях:
  - 📄 Файловая система (read_file, write_file, list_directory)
  - 💻 Консоль (execute_command)
  - 🌐 Веб (web_search)
  - 🧮 Математика (calculate)
  - 🐍 Python (python_exec, pip_install)
  - 🖼️ Медиа (image_info)
- Уровни риска: безопасные 🟢, требуют проверки 🟡, опасные 🔴
- Включение/выключение инструментов
- Колбэки на изменения

### Менеджер агентов (AgentManager)
- Главный агент (создается автоматически)
- До 10 суб-агентов с ролями:
  - 👑 MAIN - главный агент
  - 🔍 RESEARCHER - исследователь
  - 💻 CODER - программист
  - ✍️ WRITER - писатель
  - ✅ REVIEWER - ревьювер
  - 🤖 CUSTOM - пользовательская роль
- Отслеживание статусов (idle/working/waiting/error/stopped)
- Управление задачами и статистикой
- Процент успешности задач

### Настройки
- URL LM Studio сервера
- Тема оформления (dark/light/system)
- Авто-подключение при запуске
- Логирование

## 🔗 Интеграция с LM Studio

Для реальной работы требуется:
1. Установить [LM Studio](https://lmstudio.ai/)
2. Загрузить модель
3. Включить сервер на `localhost:1234`
4. Подключиться через кнопку в приложении

## 📝 Логи

Приложение ведет логирование в:
- Консоль (stdout)
- Файл `deepagents.log`
