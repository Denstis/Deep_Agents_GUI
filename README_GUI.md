# DeepAgents GUI

Полнофункциональное GUI приложение на customtkinter для работы с DeepAgents через LM Studio server и локальные модели.

## Особенности

- **Чат интерфейс** - Современный темный UI с пузырьками сообщений
- **Интеграция с DeepAgents** - Полная поддержка библиотеки deepagents от LangChain
- **LM Studio поддержка** - Работа только с локальными моделями через LM Studio сервер
- **Инструменты файловой системы**:
  - Чтение файлов
  - Запись файлов
  - Просмотр содержимого директорий
  - Выполнение безопасных shell команд
- **Настройки подключения** - Настройка URL LM Studio, выбор модели, температура
- **Индикатор соединения** - Визуальное отображение статуса подключения к LM Studio
- **История разговора** - Сохранение контекста диалога
- **Асинхронная обработка** - Неблокирующий интерфейс во время выполнения запросов

## Требования

- Python 3.11+
- customtkinter
- langchain-core
- langchain-openai
- langgraph
- deepagents (опционально, рекомендуется)
- httpx
- LM Studio с загруженной моделью

## Установка

```bash
# Установить зависимости
pip install customtkinter langchain-core langchain langgraph langchain-openai httpx

# Установить deepagents (рекомендуется)
cd deepagents/libs/deepagents
pip install .
```

## Использование

### 1. Запустите LM Studio

1. Откройте LM Studio
2. Загрузите локальную модель
3. Включите локальный сервер (обычно на http://localhost:1234)

### 2. Запустите GUI

```bash
python deepagents_gui.py
```

### 3. Настройте подключение

1. Нажмите кнопку "⚙ Settings"
2. Проверьте URL LM Studio (по умолчанию http://localhost:1234)
3. При необходимости укажите имя модели
4. Настройте температуру генерации
5. Нажмите "Save"

### 4. Начните общение

Введите ваш запрос в текстовое поле и нажмите "Send" или Ctrl+Enter.

## Возможности DeepAgents

При установленной библиотеке deepagents приложение использует полную функциональность:

- **Планирование задач** - Автоматическое разбиение сложных задач
- **Управление контекстом** - Автосуммаризация длинных диалогов
- **Суб-агенты** - Делегирование задач специализированным агентам
- **Memory** - Долговременная память между сессиями
- **Skills** - Поддержка пользовательских навыков

## Безопасность

- Разрешены только безопасные read-only команды (ls, cat, head, tail, pwd, echo, find, grep)
- Запрещено выполнение команд с потенциально опасными операциями
- Все операции с файлами проверяются на существование путей

## Troubleshooting

### "Cannot connect to LM Studio"

- Убедитесь, что LM Studio запущен
- Проверьте, что модель загружена
- Проверьте URL сервера в настройках

### Ошибки импорта

```bash
# Переустановите зависимости
pip install --upgrade customtkinter langchain-core langchain langgraph
```

### Проблемы с tkinter

```bash
# Для Debian/Ubuntu
apt-get install python3-tk tk
```
