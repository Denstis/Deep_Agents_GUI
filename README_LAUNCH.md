# 🚀 Deep Agents GUI - Руководство по запуску

## Быстрый старт для пользователя

### Способ 1: Интерактивное меню (Рекомендуется)

Просто **дважды кликните** на файл `START_HERE.bat` и следуйте инструкциям на экране.

Меню предложит вам:
- Установку зависимостей
- Настройку API ключей
- Запуск тестов
- Запуск приложения

---

### Способ 2: Пошаговый ручной запуск

#### Шаг 1: Установка зависимостей
```bash
1_install.bat
```
**Что делает:**
- Проверяет наличие Python 3.10+
- Создаёт виртуальное окружение (venv)
- Устанавливает все необходимые библиотеки

**Время:** ~3-5 минут (в зависимости от скорости интернета)

---

#### Шаг 2: Настройка API ключей
```bash
3_setup_env.bat
```
**Что делает:**
- Создаёт файл `.env` с шаблоном конфигурации
- Открывает файл для редактирования
- Позволяет указать ваши API ключи

**Необходимые ключи:**
| Ключ | Обязательный | Где получить |
|------|--------------|--------------|
| `OPENAI_API_KEY` | ✅ Да | https://platform.openai.com/api-keys |
| `LANGSMITH_API_KEY` | ❌ Нет (опционально) | https://smith.langchain.com |

**Пример заполнения .env:**
```ini
# OpenAI API Key (обязательно)
OPENAI_API_KEY=sk-proj-xxxxxxxxxxxxxxxxxxxxxxxx

# LangSmith API Key (опционально, для трассировки)
LANGSMITH_API_KEY=lsv2_xxxxxxxxxxxxxxxxxxxxxxxxxxxx
LANGSMITH_TRACING=true
LANGSMITH_PROJECT=deep-agents-gui

# Рабочая директория
WORKING_DIR=./workspace_files
```

---

#### Шаг 3: Проверка системы (Тесты)
```bash
2_run_tests.bat
```
**Что делает:**
- Запускает 22 модульных теста инструментов
- Запускает 18 интеграционных тестов
- Показывает отчёт о прохождении

**Ожидаемый результат:**
```
============================== 40 passed in 8.54s ==============================
[OK] ВСЕ ТЕСТЫ ПРОЙДЕНЫ УСПЕШНО!
```

---

#### Шаг 4: Запуск приложения
```bash
4_launch_gui.bat
```
**Что делает:**
- Активирует виртуальное окружение
- Загружает переменные из `.env`
- Запускает графический интерфейс

**Что вы увидите:**
- Главное окно приложения
- Панель управления инструментами
- Окно чата с агентом
- Визуализатор графа агента

---

## Дополнительные утилиты

### Быстрый демонстрационный тест
```bash
5_quick_test.bat
```
Запускает быстрый тест без GUI для проверки работоспособности системы. Полезно для быстрой диагностики проблем.

---

### Очистка временных файлов
```bash
6_cleanup.bat
```
Удаляет:
- Кэш Python (`__pycache__`)
- Логи тестов
- Временные файлы чекпоинтов
- Тестовые директории

**Опционально:**
- Виртуальное окружение (venv)
- Файл `.env` с API ключами ⚠️

## Решение проблем

### ❌ Ошибка: "Python не найден"
**Решение:**
1. Установите Python 3.10+ с https://python.org
2. При установке отметьте галочку **"Add Python to PATH"**
3. Перезапустите терминал/командную строку

---

### ❌ Ошибка: "OPENAI_API_KEY не установлен"
**Решение:**
1. Запустите `3_setup_env.bat`
2. Получите ключ на https://platform.openai.com/api-keys
3. Вставьте ключ в файл `.env`
4. Сохраните файл и запустите `4_launch_gui.bat`

---

### ❌ Ошибка: "ModuleNotFoundError: No module named 'langchain'"
**Решение:**
1. Убедитесь, что запустили `1_install.bat`
2. Проверьте, что используете активированный venv
3. Запустите вручную: `venv\Scripts\activate && pip install -r requirements.txt`

---

### ❌ Ошибка: "All tests failed"
**Решение:**
1. Проверьте интернет-соединение
2. Убедитесь, что API ключ действителен
3. Проверьте баланс аккаунта OpenAI
4. Запустите `5_quick_test.bat` для детальной диагностики

---

### ❌ Ошибка: "GUI не запускается / пустое окно"
**Решение:**
1. Проверьте логи в консоли после запуска
2. Убедитесь, что установлен `tkinter` (входит в стандартный Python)
3. На Windows: переустановите Python с опцией "tcl/tk support"
4. На Linux: `sudo apt-get install python3-tk`
