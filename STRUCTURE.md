# DeepAgents GUI v2.0 - Структура проекта

## 📁 Архитектура

```
/workspace/
├── app.py                      # Точка входа (запуск приложения)

├── gui/                        # Новый модульный GUI пакет
│   ├── __init__.py
```

## 🎯 Ключевые улучшения

### 1. Модульная архитектура
- **Раньше**: Монолитный `deepagents_gui.py` (1,378 строк)
- **Теперь**: 5 модулей по ~200-600 строк каждый

### 2. Менеджер инструментов (`tool_manager.py`)
```python
features:
  - Регистрация/удаление инструментов
  - Включение/выключение
  - Категоризация (filesystem, web, math, python, console)
  - Уровни риска (safe, review, dangerous)
  - Колбэки на изменения
```

### 3. Менеджер агентов (`agent_manager.py`)
```python
features:
  - Создание суб-агентов с ролями
  - Отслеживание статусов (idle, working, waiting, error)
  - Управление задачами
  - Статистика производительности
  - Лимит агентов (max 10)
```

### 4. Компоненты UI (`components.py`)
```python
widgets:
  - StatusIndicator      # Индикатор статуса с цветом
  - ToolCard             # Карточка инструмента
  - AgentCard            # Карточка агента
  - ProgressBar          # Прогресс бар
  - InfoPanel            # Панель информации
```

### 5. Главное окно (`main_window.py`)
```
interface_layout:
  - Header: Логотип, статус подключения, настройки
  - Sidebar: Навигация (5 вкладок), быстрые действия
  - Main Area: Динамический контент
    * 💬 Чат - общение с агентом
    * 🔧 Инструменты - управление инструментами
    * 🤖 Агенты - менеджер агентов
    * 📊 Статистика - карточки метрик
    * 📝 Логи - системные логи
  - Statusbar: Текущий статус, информация
```

## 🎨 Дизайн системы

### Цветовая схема
```python
colors = {
    "bg_primary": "#1a1a2e",      # Темно-синий фон
    "bg_secondary": "#16213e",    # Светлее для панелей
    "bg_card": "#0f3460",         # Фон карточек
    "accent": "#e94560",          # Акцентный цвет
    "text_primary": "#ecf0f1",    # Белый текст
    "text_secondary": "#95a5a6",  # Серый текст
    "success": "#2ecc71",         # Зеленый
    "warning": "#f39c12",         # Оранжевый
    "error": "#e74c3c",           # Красный
}
```

### Роли агентов
- 👑 **Main** - Главный агент
- 🔍 **Researcher** - Поиск информации
- 💻 **Coder** - Написание кода
- ✍️ **Writer** - Тексты
- ✅ **Reviewer** - Проверка
- 🤖 **Custom** - Пользовательская роль

### Уровни риска инструментов
- 🟢 **Safe** - Безопасные (чтение файлов, поиск)
- 🟡 **Review** - Требуют проверки (запись, команды)
- 🔴 **Dangerous** - Опасные (удаление, выполнение)
