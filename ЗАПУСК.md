## ⚙️ Настройка для других платформ

### OpenAI API (платно)
1. Получите ключ: https://platform.openai.com/api-keys
2. Откройте `.env`
3. Замените:
   ```
   OPENAI_API_KEY=sk-ваш-ключ
   BASE_URL=https://api.openai.com/v1
   ```

### Ollama (локально)
1. Установите: https://ollama.ai/
2. Запустите: `ollama pull llama3 && ollama serve`
3. В `.env` установите:
   ```
   BASE_URL=http://localhost:11434/v1
   OPENAI_MODEL=llama3
   ```

## ❓ Решение проблем

### "ModuleNotFoundError: No module named 'deepagents'"
```bash
venv\Scripts\activate
pip install -e .
```

### "Connection refused" (LM Studio)
1. Проверьте, что LM Studio запущен
2. Сервер активен (кнопка "Stop Server" видна)
3. Порт в `.env` совпадает с портом в LM Studio

### "40 tests failed"
1. Удалите папку `venv`
2. Запустите `1_install.bat` заново
3. Запустите `2_run_tests.bat`

### LangSmith "403 Forbidden"
Это нормально! LangSmith отключён по умолчанию.
